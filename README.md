# mtm6407_Contact-Book


A simple Vue 3 + Vite contact book app for managing contacts in the browser using Local Storage.

## Features

- View all contacts
- Add a new contact
- Edit contact information
- Delete a contact
- View individual contact details
- Responsive and styled using UIkit (or your framework)

## Technologies Used

- Vue 3 (Composition API + `<script setup>`)
- Vue Router 4
- Vite
- Local Storage for persistent data

## How to Run

1. Clone the repo  
   `git clone https://github.com/drol0039/mtm6407_Contact-Book.git`

2. Install dependencies  
   `cd contact-app`  
   `npm install`

3. Start the dev server  
   `npm run dev`

4. Visit  
   [http://localhost:5173](http://localhost:5173)

## Notes

- All data is stored in the browser’s localStorage.
- You can clear your contacts by clearing browser storage.
- Fully self-contained, no database or backend required.
